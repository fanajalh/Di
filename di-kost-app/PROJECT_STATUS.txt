                         🚀 HOW TO USE THE APP
================================================================================

1. QUICK START (3 Steps):
   npm install --legacy-peer-deps
   npm start
   Scan QR code atau press 'a'/'i'/'w'

   npm start
   Make changes → Hot reload otomatis

3. BUILD APK:
   npm install -g eas-cli

   See BUILD.md untuk detailed instructions

================================================================================
                        🆘 TROUBLESHOOTING QUICK REFERENCE
================================================================================

Issue: Metro bundler failed to start
→ Solution: npm start -- --reset-cache

Issue: Port already in use
→ Solution: npm start -- --port 8081

Issue: Cannot find expo
→ Solution: npm install -g expo-cli && npm install --legacy-peer-deps

Issue: App stuck at loading
→ Solution: Check terminal for errors, reload app, npm start -- --reset-cache

Issue: QR code not showing
→ Solution: Make terminal bigger, or copy URL from terminal

Issue: Stuck with dependencies
→ Solution: rm -rf node_modules package-lock.json && npm install --legacy-peer-deps
